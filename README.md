## Deployment
Both backend and frontend is deployed on azure.
Backend - http://api-limehome-task.azurewebsites.net/api/v1/
Frontend - https://limehome-task.azurewebsites.net/


## Requirements
 - python3.5+
 - docker, docker-compose
 - node

## To run locally
* Backend

```$ docker-compose up --build ```

## Tests

```$ make test```
